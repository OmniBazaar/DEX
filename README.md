# DEX

Decentralized Exchange (DEX) for OmniBazaar/OmniCoin, based on dYdX V4 with COTI V2 privacy integration.

## Features

- Order book and AMM trading
- OmniCoin and COTI V2 privacy integration
- Staking, governance, and rewards
- Advanced trading tools (margin, futures, options)
- Security and risk management
- Comprehensive test suite

## Getting Started

1. Clone the repository
1. Install dependencies
1. Configure environment variables
1. Start the development server

### Prerequisites

- Node.js >= 16
- npm >= 8
- TypeScript
- Hardhat
- OpenZeppelin contracts
- COTI V2 SDK

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/OmniBazaar/DEX.git
   cd DEX
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Build the project:

   ```bash
   npm run build
   ```

4. Run tests:

   ```bash
   npm test
   ```
